# ✂️ The Golden Shear

A premium barbershop website with a sophisticated dark/gold aesthetic and bilingual support.

## ✨ Features

- Responsive design (mobile + desktop)
- Smooth scrolling navigation
- Service cards with hover effects
- Image gallery grid layout
- Booking form with service selection
- **🌐 English/German language toggle** – instant translation without page reload

## 🌍 Language Toggle

The website includes a built-in language switcher that translates all visible content between English and German. Click the **🇩🇪 Deutsch** button in the navigation to switch languages – no page refresh required.
